# SupplyChainAnalytics
#Supply Chain Analytics

This project focuses on analyzing supply chain and inventory data using Python.
The goal is to understand inventory trends, identify patterns, and gain insights that can help improve supply chain decision-making.

#Project Overview

In this project, I:

Loaded and cleaned supply chain data

Performed exploratory data analysis (EDA)

Analyzed inventory levels and trends

Created visualizations to understand patterns

Generated insights to support better inventory management

#Tools Used

Python

Jupyter Notebook

Pandas

NumPy

Matplotlib / Seaborn

#Conclusion

This project showcases exploratory data analysis and insights for supply chain and inventory analytics. The results highlight patterns that can support better decision-making in managing inventory levels and improving supply chain efficiency.
